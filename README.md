# ZHS-Awesome-Cube-Simulator
> 좀비고 어썸큐브 시뮬레이터

좀비고 내에 존재하는 어썸큐브 뽑기를 시뮬레이션할 수 있는 간단한 앱입니다.
<br/><br/>

## 사용 예제
뽑기 버튼 누르면 세가지 스킨이 무작위로 뜨며, 이 중 한가지를 고를 수 있습니다. 선택한 스킨은 획득한 스킨 목록에서 밝게 보이게 됩니다.
